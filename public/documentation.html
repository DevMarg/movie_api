The documentation for the server-side component of a “movies” web application.

The web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, find the information about movies, and create a list of their favorite movies.

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie.
