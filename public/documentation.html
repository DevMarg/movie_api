Documentation for the server-side component of a “movies” web application.

Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies /movies GET None A JSON array of movie objects
Allow new users to register /users POST A JSON object with user details. Example: { "Username" : "testuser1", "Password" : "password1", "Email" : "testuser1@gmail.com", "Birthday" : "10/10/90" } A JSON object with user details including UUID. Example: { "Username": "testuser1", "Password": "password1", "Email": "testuser1@gmail.com", "Birthday": "1990-10-09T23:00:00.000Z", "FavoriteMovies": [], "_id": "6670390ffb6f33d439ee8bb8", "__v": 0 }
Return user info by username /users/:Username GET None A JSON object with user details including UUID. Example: { "_id": "6670390ffb6f33d439ee8bb8", "Username": "testuser1", "Password": "password1", "Email": "testuser1@gmail.com", "Birthday": "1990-10-09T23:00:00.000Z", "FavoriteMovies": [], "__v": 0 }
Allow users to update their information by username /users/:Username PUT A JSON object with new user information. Example: { "Username" : "updtestuser1", "Password" : "updpassword1", "Email" : "updtestuser1@gmail.com", "Birthday" : "10/10/90" } A JSON object with updated user information. Example: { "_id": "6670390ffb6f33d439ee8bb8", "Username": "updtestuser1", "Password": "updpassword1", "Email": "updtestuser1@gmail.com", "Birthday": "1990-10-09T23:00:00.000Z", "FavoriteMovies": [], "__v": 0 }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None A JSON object with updated user data including favorite movies. Example: { "_id": "6670390ffb6f33d439ee8bb8", "Username": "updtestuser1", "Password": "updpassword1", "Email": "updtestuser1@gmail.com", "Birthday": "1990-10-09T23:00:00.000Z", "FavoriteMovies": [ "665db63e036f555077cdcdf6" ], "__v": 0 }
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE None A JSON object with updated user data after removing the movie. Example: { "_id": "6670390ffb6f33d439ee8bb8", "Username": "updtestuser1", "Password": "updpassword1", "Email": "updtestuser1@gmail.com", "Birthday": "1990-10-09T23:00:00.000Z", "FavoriteMovies": [], "__v": 0 }
Delete a user by username /users/:Username DELETE None A text message indicating success or failure
